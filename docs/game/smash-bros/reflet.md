## コンボ

コンボ|%帯|ダメージ|備考
--|--|--|--
下投げ → 上強 0%付近|ー|コンボ後は密着状態を作り、ルフレには不利な場面となるため、密着が強い相手にはあまり使わない
サンダー → DA|0%〜中%|ー|幅広い%帯で使用可能。リスクが低く、その後の展開も良い。
下投げ → 空N|ー|ー|着地狩り耐性のある相手によく使う
下投げ → 空上|ー|ー|着地狩り耐性の無い相手によく使う
空前→サンダーソード|0〜30%|ー|ー|
ギガファイヤー → |0〜高%|ー|ー|
